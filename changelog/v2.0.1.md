## API

| Tag | Change |
|---:|---|
| ![feature] | <details><summary>SQLite-based balance history for graph visualization</summary><br>Periodic snapshots of all bank account balances are now stored in a per-world SQLite database (`data/BankSystem/Database/banksystem.db`). The `BalanceHistoryManager` provides filtered query and deletion APIs by date range, account number, and item type â€” ready for future graph UI integration. Snapshot interval is configurable via `BALANCE_SNAPSHOT_INTERVAL_MINUTES` server setting (default: 1 min). Master server only.</details> |
| ![feature] | <details><summary>Extensible per-user custom data on the User object</summary><br>Added `CompoundTag customData` field to `User` for storing arbitrary per-user settings. | ![feature] | <details><summary>Item price provider API for total wealth tracking</summary><br>New `ItemPriceProvider` functional interface and `setPriceCurrencyItem()` on `BankSystemAPI` allow external mods (e.g. StockMarket) to supply item market prices. See `documentation/developer-guide/ItemPriceProviderAPI.md`.</details> |
