## Overview

This version introduces multi-server improvements, code organization enhancements, and bug fixes.

## ✨ Added

### Multi-Server Features
- **Added** - List of trusted slave servers
  - Server administrators can now maintain a whitelist of trusted slave servers
  - Improves security in multi-server banking networks
  - Prevents unauthorized server connections
  - Git commit: 336bc9a

## 🔧 Changed

### Code Organization
- **Changed** - Moved Minecraft-related code into dedicated package
  - Reorganized code structure for better maintainability
  - Separated Minecraft-specific implementations from core logic
  - Improved package hierarchy: blocks, items, entities, commands now in minecraft/ package
  - Git commit: 339bb1b

### UI/UX
- **Improved** - Various UI enhancements
  - General user interface improvements
  - Better visual feedback
  - Git commit: 0ed7698
